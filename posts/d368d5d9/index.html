맥 흔한 한영키로 한영 전환하게 설정 (우측 alt 키)

윈도우 사용자가 흔히 사용하는 한영키(우측 alt 키)를 맥에서 사용하는 방법이다. 일반적인 윈도우용 키보드 기준 설명이므로 키보드의 스캔코드 차이에 따라 다르게 동작할 가능성도 있다. (그렇다 하더라도 Karabiner 란 프로그램은 인터넷을 통해 다양한 커스텀 룰을 검색하여 적용할 수 있다)

  • Karabiner-Elements 설치

  • Karabiner-Elements 열기

  • Complex Modification 메뉴 선택

  • Add-predefined rule 버튼 클릭

  • 상단 Import more rules from the internet (Open a web browser) 선택

    • 웹브라우저를 통해 한 사이트가 띄워지게 됨

  • 검색창에 [korean] 타이핑 후 엔터

  • R_ALT to switch input languages as it is in Windows (especially Korean) 찾기

    • 없다면 alt korean 등으로 검색

  • 우측 Import 버튼 클릭

  • Karabiner-Elements 앱 다시 띄움

  • 방금 Import 된 것 Enable

이렇게 하면 우측 alt 키 (윈도우 101키 레이아웃에서 한영키에 해당) 를 통해 한영 변환이 가능하다. 키를 떼었을때 전환되는 것이 아니라 키를 눌렀을때 전환되므로 빠르게 전환된다. 윈도우에서 빠른 한영 전환을 통한 타이핑에 익숙해진 경우 만족할 수 있다. 물론 그럼에도 불구하고 윈도우 만큼의 즉각적인 반응은 아니긴 하지만 다른 방법보다는 훨씬 만족스럽다.
